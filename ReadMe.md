**Ответы на вопросы**

**1.Какие основные типы памяти используются в OpenCL?**

В OpenCL используется иерархическая модель памяти:

* Global memory (глобальная память)

- Доступна всем рабочим элементам (work-items).

- Имеет наибольший объём, но относительно высокую задержку.

- Используется для хранения входных и выходных данных (например, массивы и матрицы).

* Local memory (локальная память)

- Общая для всех work-items внутри одной рабочей группы (work-group).

- Быстрее глобальной памяти.

- Часто используется для оптимизации (кэширование данных, tile-умножение матриц).

* Private memory (приватная память)

- Доступна только одному work-item.

- Самая быстрая память.

- Обычно реализуется регистрами (локальные переменные ядра).

* Constant memory (константная память)

- Только для чтения.

- Оптимизирована для случаев, когда все work-items читают одни и те же данные.

- Используется для констант и параметров ядра.
